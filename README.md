**Arcane Eye** <br>
A Screenshot Snip and Auto-Display tool for In-Person TTRPG (Table Top RPG Gaming) where a TV is used as a tabletop.<br><br>
Written in Python 3.12 and uses the QT Framework to stay compatible with Linux, Windows and Mac.<br>
Dependencies:
* PySide6
* Pynput

I have personally tested under:<br>
* Windows 10<br>
* Linux Arch/Endeavouros KDE<br><br>

If others give me feedback I will maintain a list of tested/verified OSes here
